# 算法学习笔记

这个项目用于记录算法学习的内容，包括算法模板和学习笔记。

## 目录结构

```
algo/
├── docs/                     # 文档笔记
│   ├── data-structures/      # 数据结构笔记
│   └── algorithms/           # 算法笔记
├── templates/                # 算法模板（按语言分类）
│   ├── python/
│   └── swift/
└── problems/                 # 练习题
    ├── leetcode/
    └── lanqiao/
```

## 使用说明

- `docs/` - 存放算法和数据结构的学习笔记
- `templates/` - 存放各种语言的算法模板代码
- `problems/` - 存放练习题的解答

## 学习进度

- **LeetCode**: 7 题
- **蓝桥杯**: 9 题
- **算法模板**: 快速幂、并查集、欧拉筛、质数工具、字典树（Python, Swift）
- **数据结构文档**: 并查集、字典树
- **算法文档**: 快速幂、欧拉筛

## 项目特色

- 题解文档通过相对路径链接到模板代码，实现代码复用
- 每个平台都有 README.md 索引文件，按算法类型分类
- 支持多种编程语言的模板实现
