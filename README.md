# 算法学习笔记

这个项目用于记录算法学习的内容，包括算法模板和学习笔记。

## 目录结构

```
algo/
├── docs/                     # 文档笔记
│   ├── data-structures/      # 数据结构笔记
│   └── algorithms/           # 算法笔记
├── templates/                # 算法模板（按语言分类）
│   ├── python/
│   ├── java/
│   ├── cpp/
│   ├── go/
│   └── javascript/
└── problems/                 # 练习题
    ├── leetcode/
    └── lanqiao/
```

## 使用说明

- `docs/` - 存放算法和数据结构的学习笔记
- `templates/` - 存放各种语言的算法模板代码
- `problems/` - 存放练习题的解答
