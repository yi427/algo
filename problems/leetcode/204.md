# 204. 计数质数

## [题目描述](https://leetcode.cn/problems/count-primes/)

给定整数 n，返回所有小于非负整数 n 的质数的数量。

**示例 1：**
- 输入：n = 10
- 输出：4
- 解释：小于 10 的质数一共有 4 个，它们是 2, 3, 5, 7。

**示例 2：**
- 输入：n = 0
- 输出：0

**示例 3：**
- 输入：n = 1
- 输出：0

**提示：**
- 0 <= n <= 5 × 10⁶

## 解题思路

使用欧拉筛（线性筛）算法高效地筛选出所有质数。

**核心思想：**
- 每个合数只会被它的最小质因数筛掉一次
- 时间复杂度为 O(n)，是最优的质数筛选算法

**算法步骤：**
1. 初始化标记数组，假设所有数都是质数
2. 从 2 开始遍历，如果当前数是质数，加入质数列表
3. 用已找到的质数去筛选合数
4. 关键优化：当 i % p == 0 时停止，保证每个合数只被最小质因数筛一次

## 代码实现

```python
class Solution:
    def euler_sieve(self, n: int) -> list[int]:
        is_prime = [True] * (n + 1)
        primes = []

        for i in range(2, n + 1):
            if is_prime[i]:
                primes.append(i)

            for p in primes:
                if i * p > n:
                    break
                is_prime[i * p] = False
                if i % p == 0:
                    break

        return primes

    def countPrimes(self, n: int) -> int:
        return len(self.euler_sieve(n - 1))
```

## 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)
