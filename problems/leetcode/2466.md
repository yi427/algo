# 2466. Count Ways To Build Good Strings

## [题目描述](https://leetcode.com/problems/count-ways-to-build-good-strings/description/)

给定整数 `low`、`high`、`zero`、`one`。初始有一个空字符串，每次可以在末尾追加：

- `zero` 个 `'0'`
- `one` 个 `'1'`

如果字符串长度在区间 `[low, high]` 内，则称其为好字符串。返回好字符串的数量，答案对 `10^9 + 7` 取模。

## 解题思路

使用记忆化递归。

设 `dfs(i)` 表示构造长度恰好为 `i` 的字符串方案数，则有：

```text
dfs(i) = dfs(i - zero) + dfs(i - one)
```

边界：
- `i < 0` 时返回 `0`
- `i == 0` 时返回 `1`（空串是一种构造方式）

最终答案为：

```text
sum(dfs(i)) for i in [low, high]
```

## 代码实现

```python
from functools import cache


class Solution:
    def countGoodStrings(self, low: int, high: int, zero: int, one: int) -> int:
        M = int(1e9) + 7

        @cache
        def dfs(i: int) -> int:
            if i < 0:
                return 0
            if i == 0:
                return 1

            return (dfs(i - zero) + dfs(i - one)) % M

        res = 0
        for i in range(low, high + 1):
            res = (res + dfs(i)) % M
        return res
```

## 复杂度分析

- 时间复杂度：`O(high)`，每个状态最多计算一次
- 空间复杂度：`O(high)`，用于递归栈和缓存
