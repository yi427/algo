# 50. Pow(x, n)

## [题目描述](https://leetcode.cn/problems/powx-n/)

实现 `pow(x, n)`，即计算 `x` 的整数 `n` 次幂函数。

**示例：**
- 输入：x = 2.00000, n = 10
- 输出：1024.00000

- 输入：x = 2.00000, n = -2
- 输出：0.25000

## 解题思路

使用快速幂算法，核心思想是将指数 n 进行二进制分解：
- 当 n 为偶数时：x^n = (x^2)^(n/2)
- 当 n 为奇数时：x^n = x * x^(n-1)

通过位运算优化，每次将指数右移一位，底数平方，时间复杂度从 O(n) 降到 O(log n)。

## 代码实现

### Python 实现（递归）

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0:
            return 1.0 / self.myPow(x, -n)
        if n == 0:
            return 1.0

        if n & 1:
            return x * self.myPow(x, n - 1)
        return self.myPow(x * x, n // 2)
```

### Swift 实现（迭代）

```swift
class Solution {
    func myPow(_ x: Double, _ n: Int) -> Double {
        var base = x, power = n.magnitude

        var res: Double = 1.0
        while power > 0 {
            if power & 1 == 1 {
                res *= base
            }
            base *= base
            power >>= 1
        }
        return n < 0 ? 1.0 / res : res
    }
}
```


## 复杂度分析

- 时间复杂度：O(log n)
- 空间复杂度：O(1)

## 注意事项

- 处理负指数：将 x 转换为 1/x，n 转换为 -n
- 处理 n = 0 的边界情况
