# 70. 爬楼梯

## [题目描述](https://leetcode.cn/problems/climbing-stairs/)

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例 1：**
- 输入：`n = 2`
- 输出：`2`

**示例 2：**
- 输入：`n = 3`
- 输出：`3`

**提示：**
- `1 <= n <= 45`

## 解题思路

这道题本质上是一个斐波那契型动态规划问题。

设 `f(i)` 表示爬到第 `i` 阶的方法数，那么最后一步只有两种情况：
- 从第 `i - 1` 阶爬 `1` 步上来
- 从第 `i - 2` 阶爬 `2` 步上来

所以状态转移方程为：

```text
f(i) = f(i - 1) + f(i - 2)
```

边界条件：
- `f(0) = 1`
- `f(1) = 1`

下面给出三种写法：记忆化递归、递推和状态压缩。

## 代码实现

### 解法一：记忆化递归

```python
from functools import cache


class Solution:
    def climbStairs(self, n: int) -> int:
        @cache
        def dfs(i: int) -> int:
            if i <= 1:
                return 1
            return dfs(i - 1) + dfs(i - 2)

        return dfs(n)
```

**说明：**
- `dfs(i)` 表示爬到第 `i` 阶的方法数
- 使用 `cache` 避免重复计算子问题

### 解法二：递推

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        dp = [0] * (n + 1)
        dp[0] = 1

        def get(j: int) -> int:
            return dp[j] if j >= 0 else 0

        for i in range(1, n + 1):
            dp[i] = get(i - 1) + get(i - 2)

        return dp[n]
```

**说明：**
- `dp[i]` 表示爬到第 `i` 阶的方法数
- `get` 用于统一处理 `i - 2 < 0` 的边界情况

### 解法三：状态压缩

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        a, b = 0, 1
        res = 0

        for _ in range(1, n + 1):
            res = a + b
            a, b = b, res

        return res
```

**说明：**
- 由于当前状态只依赖前两个状态，可以将 `dp` 数组压缩成两个变量
- `a` 和 `b` 分别表示前两个状态的方法数

## 复杂度分析

### 记忆化递归

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`，用于递归栈和缓存

### 动态规划

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

### 滚动数组优化

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 总结

这道题是动态规划入门题，三种写法都基于同一个状态转移方程：

- 问题具有重复子问题
- 当前状态只依赖前两个状态

其中：
- 记忆化递归更贴近状态定义
- 递推写法更直观
- 状态压缩在保持 `O(n)` 时间复杂度的同时，将空间复杂度优化到 `O(1)`
