# 2266. Count Number of Texts

## [题目描述](https://leetcode.com/problems/count-number-of-texts/)

给定一个仅由数字 `2` 到 `9` 组成的字符串 `pressedKeys`，表示手机按键的按压序列。每个数字对应若干字母（`7` 和 `9` 对应 4 个字母，其余对应 3 个字母）。求可以还原出的短信总数，答案对 `10^9 + 7` 取模。

## 解题思路

使用记忆化递归。

设 `dfs(i, r)` 表示处理到下标 `i` 时、当前数字还允许连续选择 `r` 次的方案数。转移分两种情况：

- 必须切断连续段（`r == 1` 或与前一位数字不同），进入上一位并重置可选次数
- 继续当前连续段，或切断后重置可选次数，两者求和

通过缓存避免重复计算。

## 代码实现

```python
from functools import cache


class Solution:
    def countTexts(self, pressedKeys: str) -> int:
        M = int(1e9) + 7
        f = {
            "2": 3, "3": 3, "4": 3, "5": 3, "6": 3, "7": 4, "8": 3, "9": 4,
        }
        n = len(pressedKeys)

        @cache
        def dfs(i: int, r: int) -> int:
            if i == 0:
                return 1

            if r == 1 or pressedKeys[i] != pressedKeys[i - 1]:
                return dfs(i - 1, f[pressedKeys[i - 1]]) % M
            return (dfs(i - 1, r - 1) + dfs(i - 1, f[pressedKeys[i - 1]])) % M

        return dfs(n - 1, f[pressedKeys[n - 1]]) % M
```

## 复杂度分析

- 时间复杂度：`O(n)`，状态数与下标线性相关（每个位置的 `r` 取值范围常数级）
- 空间复杂度：`O(n)`，用于递归栈和缓存
