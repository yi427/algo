# 377. Combination Sum IV

## [题目描述](https://leetcode.com/problems/combination-sum-iv/)

给定一个由不同整数组成的数组 `nums`，以及一个目标整数 `target`，请返回可以凑成 `target` 的有序组合总数。

## 解题思路

使用记忆化递归。

设 `dfs(t)` 表示凑出和为 `t` 的组合数。

- 当 `t == 0` 时，表示找到一种合法组合，返回 `1`
- 当 `t < 0` 时，表示当前路径无效，返回 `0`
- 枚举每个数字 `nums[i]`，累加 `dfs(t - nums[i])`

最终答案是 `dfs(target)`。

## 代码实现

```python
from functools import cache
from typing import List


class Solution:
    def combinationSum4(self, nums: List[int], target: int) -> int:
        n = len(nums)

        @cache
        def dfs(t: int) -> int:
            if t == 0:
                return 1
            if t < 0:
                return 0

            res = 0
            for i in range(n):
                res += dfs(t - nums[i])

            return res

        return dfs(target)
```

## 复杂度分析

- 时间复杂度：`O(target * n)`，每个 `t` 最多计算一次，每次遍历 `nums`
- 空间复杂度：`O(target)`，用于递归栈和缓存
