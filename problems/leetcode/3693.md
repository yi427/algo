# 3693. Climbing Stairs II

## [题目描述](https://leetcode.com/problems/climbing-stairs-ii/description/)

给定楼梯阶数 `n` 和数组 `costs`。到达第 `i` 阶需要支付 `costs[i]`，每次可以向上走 `1`、`2` 或 `3` 阶，并且本次步长 `step` 还会产生 `step * step` 的额外代价。求到达顶层的最小总代价。

## 解题思路

使用记忆化递归。

设 `dfs(i)` 表示到达第 `i` 阶的最小代价。转移时枚举上一步的步长 `step in {1,2,3}`：

```text
dfs(i) = min(dfs(i - step) + step^2 + costs[i])
```

边界：
- `i < 0` 时返回 `0`
- 最终答案为 `dfs(n - 1)`

## 代码实现

```python
from functools import cache
from typing import List


class Solution:
    def climbStairs(self, n: int, costs: List[int]) -> int:
        @cache
        def dfs(i: int) -> int:
            if i < 0:
                return 0

            res = float("inf")
            for step in range(1, 3 + 1):
                res = min(res, dfs(i - step) + step * step + costs[i])
            return res

        return dfs(n - 1)
```

## 复杂度分析

- 时间复杂度：`O(n)`，每个状态最多枚举 3 种步长
- 空间复杂度：`O(n)`，用于递归栈和缓存
