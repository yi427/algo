# 421. 数组中两个数的最大异或值

## [题目描述](https://leetcode.cn/problems/maximum-xor-of-two-numbers-in-an-array/)

给你一个整数数组 nums，返回 nums[i] XOR nums[j] 的最大运算结果，其中 0 ≤ i ≤ j < n。

**示例 1：**
- 输入：nums = [3,10,5,25,2,8]
- 输出：28
- 解释：最大运算结果是 5 XOR 25 = 28

**示例 2：**
- 输入：nums = [14,70,53,83,49,91,36,80,92,51,66,70]
- 输出：127

**提示：**
- 1 <= nums.length <= 2 × 10⁵
- 0 <= nums[i] <= 2³¹ - 1

## 解题思路

本题使用二进制字典树（Trie）来高效地找到最大异或值。

**核心思想：**
- 将每个数字的二进制表示插入到 Trie 中
- 对于每个数字，在 Trie 中贪心地选择相反的位来最大化异或结果
- 从高位到低位遍历，优先选择不同的位

**算法步骤：**
1. 构建二进制 Trie，将所有数字插入
2. 对每个数字，从最高位开始在 Trie 中查找
3. 每一位尽量选择相反的位（0选1，1选0）来最大化异或值
4. 如果相反的位不存在，则选择相同的位
5. 记录所有数字能得到的最大异或值

## 代码实现

### Python 实现

```python
class Trie:
    def __init__(self) -> None:
        self.children: list[Trie | None] = [None, None]

    def insert(self, num: int):
        root = self

        for i in range(31, -1, -1):
            bit = (num >> i) & 1
            assert root
            if not root.children[bit]:
                root.children[bit] = Trie()
            root = root.children[bit]


class Solution:
    def findMaximumXOR(self, nums: List[int]) -> int:
        # build tree
        root = Trie()
        for num in nums:
            root.insert(num)
        # find maximum xor
        res = 0
        for num in nums:
            node, xor = root, 0
            for i in range(31, -1, -1):
                bit = (num >> i) & 1
                not_bit = 0 if bit == 1 else 1
                if node.children[not_bit]:
                    node = node.children[not_bit]
                    xor |= not_bit << i
                else:
                    node = node.children[bit]
                    xor |= bit << i
            res = max(res, xor ^ num)
        return res
```

### Swift 实现

```swift
class Trie {

    public var children: [Trie?]

    init() {
        children = Array(repeating: nil, count: 2)
    }

     func insert(_ num: Int) {
         var root = self
         for i in (0..<32).reversed() {
             let bit = (num >> i) & 1
             if root.children[bit] == nil {
                 root.children[bit] = Trie()
             }
             root = root.children[bit]!
         }
    }
}



class Solution {
    func findMaximumXOR(_ nums: [Int]) -> Int {
        var root = Trie()
        for num in nums {
            root.insert(num)
        }

        var res = 0
        for num in nums {
            var node = root
            var xor = 0
            for i in (0..<32).reversed() {
                let bit = (num >> i) & 1
                let not_bit = 1 ^ bit
                if node.children[not_bit] != nil {
                    node = node.children[not_bit]!
                    xor |= (not_bit << i)
                } else {
                    node = node.children[bit]!
                    xor |= (bit << i)
                }
            }
            res = max(res, xor ^ num)
        }
        return res
    }
}
```

## 复杂度分析

- 时间复杂度：O(n × 32) = O(n)，其中 n 是数组长度，每个数字需要处理 32 位
- 空间复杂度：O(n × 32) = O(n)，Trie 树最多存储 n × 32 个节点
