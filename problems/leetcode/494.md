# 494. Target Sum

## [题目描述](https://leetcode.com/problems/target-sum/)

给定一个非负整数数组 `nums` 和一个整数 `target`，可以在每个数字前添加 `+` 或 `-` 号，问最终表达式结果等于 `target` 的方案数有多少种。

## 解题思路

这道题可以看成一个 0/1 背包型动态规划问题。当前先使用记忆化搜索来做。

设 `dfs(i, t)` 表示从下标 `0..i` 中选择符号后，最终结果为 `t` 的方案数。对于第 `i` 个数，有两种选择：

- 给 `nums[i]` 添加 `+`
- 给 `nums[i]` 添加 `-`

所以状态转移为：

```text
dfs(i, t) = dfs(i - 1, t + nums[i]) + dfs(i - 1, t - nums[i])
```

边界：
- 当 `i < 0` 且 `t == 0` 时，说明找到一种合法方案，返回 `1`
- 当 `i < 0` 且 `t != 0` 时，返回 `0`

## 代码实现

```python
from functools import cache
from typing import List


class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        n = len(nums)

        # 从 0~i 和为 t 的方案数
        @cache
        def dfs(i: int, t: int) -> int:
            if i < 0 and t == 0:
                return 1
            if i < 0:
                return 0

            return dfs(i - 1, t + nums[i]) + dfs(i - 1, t - nums[i])

        return dfs(n - 1, target)
```

## 复杂度分析

- 时间复杂度：`O(n * S)`，其中 `S` 是可达和的状态范围
- 空间复杂度：`O(n * S)`，用于递归栈和缓存
