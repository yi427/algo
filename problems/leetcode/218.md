# 218. The Skyline Problem

## [题目描述](https://leetcode.com/problems/the-skyline-problem/)

给定若干建筑物 `buildings`，其中 `buildings[i] = [lefti, righti, heighti]` 表示第 `i` 栋建筑的左右边界和高度。请返回由这些建筑物构成的天际线关键点。

## 解题思路

这道题使用扫描线 + 懒删除堆。

把每栋建筑拆成两个事件：
- 在 `left` 位置入场，表示高度 `h` 开始生效
- 在 `right` 位置出场，表示高度 `h` 失效

扫描所有事件时，需要保证：
- 先按横坐标从小到大处理
- 同一坐标先处理入场，再处理出场
- 同为入场时先处理更高的建筑
- 同为出场时先处理更矮的建筑

同时维护一个最大堆记录当前所有有效建筑的高度。由于 Python 的堆是最小堆，因此将高度取相反数存入。对于已经出场但仍留在堆中的高度，使用计数表做懒删除。

当当前最大高度发生变化时，就产生一个新的关键点。

## 代码实现

```python
import heapq
from collections import defaultdict
from functools import cmp_to_key
from typing import List


class Solution:
    def getSkyline(self, buildings: List[List[int]]) -> List[List[int]]:
        tasks = []
        for b in buildings:
            # set the task is (c, type, h)
            tasks.append((b[0], 0, b[2]))
            tasks.append((b[1], 1, b[2]))

        def com(e1: tuple[int, int, int], e2: tuple[int, int, int]) -> int:
            # c
            if e1[0] != e2[0]:
                return -1 if e1[0] < e2[0] else 1

            # 坐标相同, 先处理入场, 再处理出场
            if e1[1] != e2[1]:
                return -1 if e1[1] < e2[1] else 1

            # 同为入场, 先处理高的
            if e1[1] == 0:
                return -1 if e1[2] > e2[2] else 1

            # 同为出场, 先处理矮的
            return -1 if e1[2] < e2[2] else 1

        tasks.sort(key=cmp_to_key(com))

        f = defaultdict(int)
        f[0] = 1
        m_q = [0]

        ans, prev = [], 0

        for c, t, h in tasks:
            if t == 0:
                heapq.heappush(m_q, -h)
                f[h] += 1
            else:
                f[h] -= 1

            while f[-m_q[0]] == 0:
                heapq.heappop(m_q)

            level = -m_q[0]
            if level != prev:
                ans.append([c, level])
                prev = level

        return ans
```

## 复杂度分析

- 时间复杂度：`O(n log n)`，排序需要 `O(n log n)`，每个事件的堆操作为 `O(log n)`
- 空间复杂度：`O(n)`，用于事件数组、计数表和堆
