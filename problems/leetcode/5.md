# 5. 最长回文子串

## [题目描述](https://leetcode.cn/problems/longest-palindromic-substring/)

给你一个字符串 s，找到 s 中最长的回文子串。

**示例 1：**
- 输入：s = "babad"
- 输出："bab"
- 解释："aba" 同样是符合题意的答案

**示例 2：**
- 输入：s = "cbbd"
- 输出："bb"

**提示：**
- 1 <= s.length <= 1000
- s 仅由英文字母和数字组成

## 解题思路

本题使用 Manacher 算法（马拉车算法）在 O(n) 时间内找到最长回文子串。

**核心思想：**
- 通过在字符间插入特殊字符 "#"，将奇偶长度回文统一处理
- 利用已知的回文信息，避免重复比较
- 维护当前最右回文边界 R 和对应的中心 C

**算法步骤：**
1. 字符串预处理：在首尾添加边界字符 "^" 和 "$"，字符间插入 "#"
2. 遍历每个位置，利用对称性初始化回文半径
3. 尝试扩展回文半径
4. 更新最右边界和中心
5. 找到最大回文半径对应的子串

## 代码实现

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        t = "#".join("^" + s + "$")
        C, R = 0, 0
        P = [0] * len(t)

        for i in range(2, len(t) - 1):
            if i < R:
                P[i] = min(P[2*C - i], R - i)
            while t[i - P[i]] == t[i + P[i]]:
                P[i] += 1

            if i + P[i] > R:
                C, R = i, i + P[i]

        c, r = 0, 0
        for i in range(len(P)):
            if P[i] > r:
                c, r = i, P[i]

        return t[c-r+1:c+r].replace("#", "")
```

## 复杂度分析

- 时间复杂度：O(n)，每个位置最多被访问两次
- 空间复杂度：O(n)，存储 P 数组和转换后的字符串
