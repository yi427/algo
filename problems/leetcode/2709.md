# 2709. 最大公约数遍历

## [题目描述](https://leetcode.cn/problems/greatest-common-divisor-traversal/)

给你一个下标从 0 开始的整数数组 nums，你可以在一些下标之间遍历。对于两个下标 i 和 j（i != j），当且仅当 gcd(nums[i], nums[j]) > 1 时，我们可以在两个下标之间遍历，其中 gcd 是两个数的最大公约数。

你需要判断 nums 数组中任意两个满足 i < j 的下标 i 和 j，是否存在若干次遍历行程从 i 遍历到 j。

如果任意满足条件的下标对都可以遍历，那么返回 true，否则返回 false。

**示例 1：**
- 输入：nums = [2,3,6]
- 输出：true
- 解释：可以从 0 到 1，从 0 到 2，从 1 到 2

**示例 2：**
- 输入：nums = [3,9,5]
- 输出：false

**示例 3：**
- 输入：nums = [4,3,12,8]
- 输出：true

**提示：**
- 1 <= nums.length <= 10⁵
- 1 <= nums[i] <= 10⁵

## 解题思路

本题使用并查集 + 质因数分解来判断图的连通性。

**核心思想：**
- 如果两个数有公共质因子，它们就可以连通
- 使用并查集维护连通性
- 通过质因数分解，将有相同质因子的数连接起来

**算法步骤：**
1. 使用欧拉筛预处理每个数的最小质因数
2. 对每个数进行质因数分解
3. 将有相同质因子的数用并查集连接
4. 最后判断是否所有数都在同一个连通分量中

## 代码实现

```python
class Dsu:
    def __init__(self, n: int) -> None:
        self._parent_or_size = [-1] * n
        self._n = n
        self._counter = n

    def is_root(self, n: int) -> bool:
        assert 0 <= n < self._n
        return self._parent_or_size[n] < 0

    def connected(self) -> int:
        return self._counter

    def size(self, node: int) -> int:
        root = self.find_set(node)
        return -self._parent_or_size[root]

    def find_set(self, node: int) -> int:
        if self.is_root(node):
            return node
        parent = self.find_set(self._parent_or_size[node])
        self._parent_or_size[node] = parent
        return parent

    def union_set(self, x: int, y: int) -> int:
        x = self.find_set(x)
        y = self.find_set(y)
        if x == y:
            return x

        if -self._parent_or_size[x] < -self._parent_or_size[y]:
            x, y = y, x

        self._parent_or_size[x] += self._parent_or_size[y]
        self._parent_or_size[y] = x
        self._counter -= 1
        return x


class Solution:
    def euler_sieve(self, n: int) -> list[int]:
        """欧拉筛预处理最小质因数"""
        spfs = list(range(n + 1))
        primes = []

        for i in range(2, n + 1):
            if spfs[i] == i:
                primes.append(i)

            for p in primes:
                if i * p > n:
                    break
                spfs[i * p] = p
                if i % p == 0:
                    break

        return spfs

    def canTraverseAllPairs(self, nums: List[int]) -> bool:
        spfs = self.euler_sieve(max(nums))
        f = {}
        dsu = Dsu(len(nums))

        for i, num in enumerate(nums):
            while num > 1:
                fac = spfs[num]
                if fac in f:
                    dsu.union_set(i, f[fac])
                while num % fac == 0:
                    num //= fac
                f[fac] = i

        return dsu.connected() == 1
```

## 复杂度分析

- 时间复杂度：O(M + n log log M + n log M)，其中 M = max(nums)
  - 欧拉筛：O(M)
  - 质因数分解：O(n log M)
- 空间复杂度：O(M + n)
