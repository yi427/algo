# 5003. 反异或01串

## [题目描述](https://www.lanqiao.cn/problems/5003/learning/)

初始有一个空的 01 串，每步操作可以将 0 或 1 添加在左侧或右侧。也可以对整个串进行反异或操作：取 s' = s ⊕ rev(s)，其中 s 是目前的 01 串，⊕ 表示逐位异或，rev(s) 代表将 s 翻转。

反异或操作最多使用一次（可以不用，也可以用一次）。

给定一个 01 串 T，问最少需要添加多少个 1 才能从一个空 01 串得到 T。

在本题中 0 可以添加任意个。

**输入格式：**
- 输入一行包含一个 01 串表示给定的 T

**输出格式：**
- 输出一行包含一个整数，表示需要最少添加多少个 1

**样例输入：**
```
00111011
```

**样例输出：**
```
3
```

**数据范围：**
- 对于 20% 的评测用例，|T| ≤ 10
- 对于 40% 的评测用例，|T| ≤ 500
- 对于 60% 的评测用例，|T| ≤ 5000
- 对于 80% 的评测用例，|T| ≤ 10⁵
- 对于所有评测用例，1 ≤ |T| ≤ 10⁶，保证 T 中仅含 0 和 1

## 解题思路

本题的核心在于理解反异或操作的性质：当对一个回文串进行反异或操作时，中心位置的字符会被消除。

**关键观察：**
- 如果存在一个回文子串，通过反异或操作可以消除其中一半的 1
- 我们需要找到包含最多 1 的回文子串
- 答案 = 总的 1 的个数 - 最大回文子串中 1 的个数 / 2

**两种解法：**
1. 中心扩散法：枚举每个可能的回文中心，向两边扩展
2. Manacher 算法 + 前缀和：O(n) 时间复杂度找到所有回文子串

## 代码实现

### 方法一：中心扩散法

```python
def spear(l, r, T):
    # (l, r)
    while l >= 0 and r < len(T) and T[l] == T[r]:
        l -= 1
        r += 1
    return T[l + 1 : r]


def solve(T: str):
    res = T.count("1")
    count_one = 0
    for i in range(len(T)):
        if T[i] == "0":
            count_one = max(count_one, spear(i - 1, i + 1, T).count("1"))
        if i > 0 and T[i] == T[i - 1]:
            count_one = max(count_one, spear(i - 2, i + 1, T).count("1"))
    return res - count_one // 2


if __name__ == "__main__":
    T = input()
    print(solve(T))
```

**复杂度分析：**
- 时间复杂度：O(n²)，枚举中心 O(n)，扩展 O(n)
- 空间复杂度：O(1)

### 方法二：Manacher 算法 + 前缀和

```python
from itertools import accumulate

def solve2(T: str) -> int:
    t = "#".join("^" + T + "$")
    nums = list(map(lambda x: 1 if x == '1' else 0, t))
    prefixs = list(accumulate(nums))

    P = [0] * len(nums)
    C, R = 0, 0

    for i in range(2, len(t) - 1):
        if i < R:
            P[i] = min(P[2*C-i], R - i)

        while t[i - P[i]] == t[i + P[i]]:
            P[i] += 1

        if i + P[i] > R:
            C, R = i, i + P[i]

    count_one = 0
    for i in range(2, len(P) - 1):
        if t[i] == "0" or t[i] == t[i - 2]:
            count_one = max(count_one, prefixs[i + P[i] - 1] - prefixs[i - P[i]])

    return T.count("1") - count_one // 2


if __name__ == "__main__":
    T = input()
    print(solve2(T))
```

**复杂度分析：**
- 时间复杂度：O(n)，Manacher 算法线性时间
- 空间复杂度：O(n)，存储 P 数组和前缀和数组
