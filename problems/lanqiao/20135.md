# 20135. 插入学生

## [题目描述](https://www.lanqiao.cn/problems/20135/learning/)

小蓝的班上有 n 名同学正在为校庆活动排练节目，他们在舞台上站成一排。若把他们看作数轴上的点，从左到右第 i 名同学的位置为 aᵢ。

为了美观，小蓝想在队伍中插入一些新同学，使得新队伍中任意三名相邻的同学 (aᵢ₋₁, aᵢ, aᵢ₊₁) 满足：

$$a_{i+1} - a_i \leq 2 \times (a_i - a_{i-1})$$

小蓝想知道，最少需要增加多少名同学，才能使整个队伍满足上述条件。

**输入格式：**
- 第一行包含一个正整数 n（1 ≤ n ≤ 10⁵）
- 第二行包含 n 个正整数 a₁, a₂, ..., aₙ（1 ≤ aᵢ ≤ 10⁸），保证 aᵢ₊₁ > aᵢ

**输出格式：**
- 输出一个整数，表示最少需要增加的同学数量

**样例输入：**
```
4
1 3 16 17
```

**样例输出：**
```
2
```

**样例说明：**
在位置 6 和 10 处插入两名同学，使得新队伍变为：1, 3, 6, 10, 16, 17，此时所有相邻三人均满足条件，因此答案为 2。

**数据范围：**
- 对于 20% 的评测用例，1 ≤ n ≤ 10³
- 对于 100% 的评测用例，1 ≤ n ≤ 10⁵，且 1 ≤ aᵢ ≤ 10⁸，aᵢ₊₁ > aᵢ

## 解题思路

对于每对相邻间隔 gap = aᵢ - aᵢ₋₁，如果 gap > 2b（b 为前一个间隔），需要插入若干个点。

**关键递推关系：**

设插入后的序列中，某段的间隔序列为 l₀ = a，lₙ = ⌈lₙ₋₁/2⌉，则：

$$l_n = \frac{a + B_{n-1}}{2^n}$$

其中 Bₙ₋₁ 是 a 的补码的低 n-1 位（即 `low_n_bit(complement(a), n-1)`）。

**前缀和公式推导（见手写推导图）：**

设 Sₙ = Σᵢ₌₀ⁿ lᵢ，通过数学归纳法可得：

$$S_n = 2a + \text{popcount}(B_{n-1}) - 1$$

其中 popcount 为二进制中 1 的个数，Bₙ₋₁ 为 a 的补码的低 n-1 位。

**Note：** lₙ 后面是无限多个 1，因此 Sₙ 中的 Bₙ₋₁ 为 lₙ 的最高位补码是 1，数学与计算机完美融合。

**算法步骤：**
1. 遍历相邻间隔，若 gap ≤ 2b 则直接更新 b = gap
2. 否则用对数估算需要插入的步数 step
3. 用二分查找确定插入后的最大间隔 b

## 代码实现

### 方法一：贪心 + 二分（朴素版）

```python
def sum_base(base: int, n: int) -> int:
    return base * (pow(2, n + 1) - 2)


def binary_search(left: int, right: int, n: int, b: int, gap: int) -> int:
    # [left, right)
    def check(num: int) -> bool:
        r = gap
        for _ in range(n - 1):
            r -= num
            num = (num + 1) // 2
        r -= num
        return r >= 0 and num <= 2 * b

    while left < right:
        mid = (left + right) // 2
        if check(mid):
            left = mid + 1
        else:
            right = mid
    return left - 1


def solve(nums) -> int:
    n = len(nums)
    if n <= 2:
        return 0
    res, b = 0, nums[1] - nums[0]

    for i in range(2, n):
        step = 1
        gap = nums[i] - nums[i-1]
        if gap <= 2 * b:
            b = gap
            continue

        while sum_base(b, step) < nums[i] - nums[i - 1]:
            step += 1
        res += step - 1
        b = binary_search(0, nums[i] - nums[i - 1] + 1, step, b, gap)
    return res


if __name__ == "__main__":
    _ = int(input())
    arr = list(map(int, input().split()))
    print(solve(arr))
```

### 方法二：贪心 + 二分（数学公式优化）

利用前缀和公式 Sₙ = 2a + popcount(Bₙ₋₁) - 1 优化计算：

```python
import math


def complement(num: int) -> int:
    return -num & (1 << num.bit_length()) - 1


def low_n_bit(num: int, n: int) -> int:
    if n == 0:
        return 0
    return num & (1 << n) - 1


def index(num: int, a: int) -> int:
    if a <= num:
        return 0
    return ((a + num - 1) // num - 1).bit_length()


def count_bits(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count


def sum_n(a: int, n: int) -> int:
    if a <= 1:
        return a
    return 2 * a + count_bits(low_n_bit(complement(a), n - 1)) - 1


def binary_search(left: int, right: int, n: int, b: int, gap: int) -> int:
    # [left, right)
    def check(num: int) -> bool:
        l, r = math.ceil(num / pow(2, n - 1)), num
        return 2 * b >= l and gap >= sum_n(r, index(1, r)) - sum_n(l, index(1, l)) + l

    while left < right:
        mid = (left + right) // 2
        if check(mid):
            left = mid + 1
        else:
            right = mid
    return left - 1


def solve(nums) -> int:
    n = len(nums)
    if n <= 2:
        return 0
    res, b = 0, nums[1] - nums[0]

    for i in range(2, n):
        gap = nums[i] - nums[i - 1]
        if gap <= 2 * b:
            b = gap
            continue

        step = math.ceil(math.log2((gap + 2 * b) / b) - 1)
        res += step - 1
        b = binary_search(0, gap + 1, step, b, gap)
    return res


if __name__ == "__main__":
    _ = int(input())
    arr = list(map(int, input().split()))
    print(solve(arr))
```

## 数学推导

**递推关系：**

已知 l₀ = a，lₙ = ⌈lₙ₋₁/2⌉，设 lₙ = (lₙ₋₁ + bₙ₋₁) / 2，其中 bᵢ ∈ {0, 1}。

由数学归纳法：

$$l_n = \frac{a + \sum_{i=0}^{n-1} 2^i b_i}{2^n} = \frac{a + B_{n-1}}{2^n}$$

**前缀和推导（方法一：直接展开）：**

设 Sₙ = Σᵢ₌₀ⁿ lᵢ，将每项代入展开：

$$S_n = a + \frac{a+B_0}{2} + \frac{a+B_1}{4} + \cdots + \frac{a+B_{n-1}}{2^n}$$

两边乘以 2ⁿ：

$$2^n S_n = 2^n a + 2^{n-1}(a+B_0) + 2^{n-2}(a+B_1) + \cdots + (a+B_{n-1})$$

$$= (2^n + 2^{n-1} + \cdots + 1)a + 2^{n-1}B_0 + 2^{n-2}B_1 + \cdots + B_{n-1}$$

$$= (2^{n+1}-1)a + \sum_{i=0}^{n-1} 2^{n-1-i} B_i$$

其中 Bᵢ = `low_n_bit(complement(a), i)` 为 a 的补码的低 i 位，展开 Bᵢ = Σⱼ₌₀ⁱ⁻¹ 2ʲ bⱼ：

$$\sum_{i=0}^{n-1} 2^{n-1-i} B_i = \sum_{i=0}^{n-1} 2^{n-1-i} \sum_{j=0}^{i-1} 2^j b_j = \sum_{j=0}^{n-2} b_j \sum_{i=j+1}^{n-1} 2^{n-1-i+j} = \sum_{j=0}^{n-2} b_j (2^{n-1} - 2^j)$$

代入化简：

$$2^n S_n = (2^{n+1}-1)a + 2^{n-1}\sum_{j=0}^{n-2} b_j - \sum_{j=0}^{n-2} 2^j b_j$$

$$= (2^{n+1}-1)a + 2^{n-1} B_{n-1} - (B_{n-1} - b_{n-1} \cdot 2^{n-1})$$

注意到 $2^{n-1} B_{n-1} + b_{n-1} \cdot 2^{n-1} \cdot 2^n$ 项，整理后：

$$S_n = 2a + \text{popcount}(B_{n-1}) - 1$$

**前缀和推导（方法二：错位相减）：**

由递推 lₙ = (lₙ₋₁ + bₙ₋₁) / 2，得 2lₙ = lₙ₋₁ + bₙ₋₁，写出方程组：

$$\begin{cases} 2l_1 = l_0 + b_0 \\ 2l_2 = l_1 + b_1 \\ \vdots \\ 2l_n = l_{n-1} + b_{n-1} \end{cases}$$

全部相加：$2(S_n - l_0) = S_n - l_n + \sum_{i=0}^{n-1} b_i$

$$S_n = 2l_0 - l_n + \sum_{i=0}^{n-1} b_i = 2a - \frac{a + B_{n-1}}{2^n} + \text{popcount}(B_{n-1})$$

当 n 足够大时 lₙ → 1（最终收敛到 1），且 Bₙ₋₁ 的最高位补码为 1，故：

$$S_n = 2a + \text{popcount}(B_{n-1}) - 1$$

其中 Bₙ₋₁ = `low_n_bit(complement(a), n-1)` 为 a 的补码的低 n-1 位。

## 复杂度分析

- **时间复杂度**：O(n log M)，其中 M = max(aᵢ)，每个间隔最多进行 O(log M) 次二分
- **空间复杂度**：O(1)
