# 3527. 阶乘的和

## 题目链接

[蓝桥杯 3527](https://www.lanqiao.cn/problems/3527/learning/)

## 题目描述

给定 n 个数 Aᵢ，问能满足 m! 为 ∑(Aᵢ!) 的因数的最大的 m 是多少。其中 m! 表示 m 的阶乘，即 1 × 2 × 3 × ⋯ × m。

**输入格式：**

输入的第一行包含一个整数 n。

第二行包含 n 个整数，分别表示 Aᵢ，相邻整数之间使用一个空格分隔。

**输出格式：**

输出一行包含一个整数表示答案。

**样例输入：**
```
3
2 2 2
```

**样例输出：**
```
3
```

**数据范围：**
- 对于 40% 的评测用例，n ≤ 5000
- 对于所有评测用例，1 ≤ n ≤ 10⁵，1 ≤ Aᵢ ≤ 10⁹

## 解题思路

本题使用贪心算法，核心思想是利用阶乘的性质：k! × k = (k+1)!

算法步骤：
1. 统计每个数字出现的次数
2. 从最小的数字开始，尝试构建更大的阶乘
3. 对于当前结果 res，检查是否有足够的 res 个数来组成 (res+1)!
4. 如果某个数字 k 的出现次数能被 (res+1) 整除，说明可以继续增加
5. 将多余的 k 合并成 k+1（利用 k! × k = (k+1)! 的性质）

## 代码实现

```python
from collections import Counter

if __name__ == '__main__':
    n = int(input())
    a = list(map(int, input().split()))

    c = Counter(a)
    res = min(c.keys())

    for k in range(res, int(1e9) + 1):
        if c[k] % (res + 1) != 0:
            break

        res += 1
        c[res] += c[k] // res

    print(res)
```

## 复杂度分析

- 时间复杂度：O(n + m)，其中 m 是最终结果
- 空间复杂度：O(n)
