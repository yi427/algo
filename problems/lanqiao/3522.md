# 3522. 互质数的个数

## 题目链接

[蓝桥杯 3522](https://www.lanqiao.cn/problems/3522/learning/)

## 题目描述

给定 a, b，求 1 ≤ x < a^b 中有多少个 x 与 a^b 互质，由于答案可能很大，你只需要输出答案对 998244353 取模的结果。

**输入格式：**
输入一行包含两个整数 a, b，用一个空格分隔。

**输出格式：**
输出一行包含一个整数表示答案。

**样例输入 1：**
```
2 5
```

**样例输出 1：**
```
16
```

**样例输入 2：**
```
12 7
```

**样例输出 2：**
```
11943936
```

**数据范围：**
- 对于 30% 的评测用例，ab ≤ 10^6
- 对于 70% 的评测用例，a ≤ 10^6, b ≤ 10^9
- 对于所有评测用例，1 < a ≤ 10^9, 1 ≤ b ≤ 10^18

## 解题思路

本题求小于 a^b 且与 a^b 互质的数的个数，即求欧拉函数 φ(a^b)。

根据欧拉函数的性质：
- 若 n = p1^k1 * p2^k2 * ... * pm^km（质因数分解）
- 则 φ(n) = n * ∏(1 - 1/pi) = n * ∏(pi - 1) / ∏pi

对于 a^b：
- 设 a = p1^k1 * p2^k2 * ... * pm^km
- 则 a^b = p1^(k1*b) * p2^(k2*b) * ... * pm^(km*b)
- φ(a^b) = a^b * ∏(pi - 1) / ∏pi = a^(b-1) * ∏(pi - 1)

算法步骤：
1. 对 a 进行质因数分解，得到所有不同的质因子
2. 计算 a^(b-1) mod 998244353（使用快速幂）
3. 对每个不同的质因子 p，结果乘以 (p-1)
4. 最终结果取模

## 代码实现

### 方法一：使用 set 去重

```python
def get_prime_factors(n):
    factors = []
    d = 2
    temp = n
    while d * d <= temp:
        while temp % d == 0:
            factors.append(d)
            temp //= d
        d += 1
    if temp > 1:
        factors.append(temp)
    return factors


if __name__ == "__main__":
    a, b = list(map(int, input().split()))
    mod = 998244353
    factors = get_prime_factors(a)

    visited = set()

    res = pow(a, b - 1, mod)
    for fac in factors:
        res *= fac - 1 if fac not in visited else fac
        res %= mod
        visited.add(fac)

    print(res % mod)
```

### 方法二：使用 Counter 统计

```python
from collections import Counter


def get_prime_factors(n):
    factors = []
    d = 2
    temp = n
    while d * d <= temp:
        while temp % d == 0:
            factors.append(d)
            temp //= d
        d += 1
    if temp > 1:
        factors.append(temp)
    return factors


if __name__ == "__main__":
    a, b = list(map(int, input().split()))
    mod = 998244353
    factors = get_prime_factors(a)

    counts = Counter(factors)

    res = pow(a, b - 1, mod)
    for fac in set(factors):
        res = res * pow(fac, counts[fac] - 1, mod) % mod * (fac - 1) % mod

    print(res)
```

## 复杂度分析

- 时间复杂度：O(√a + log b)
- 空间复杂度：O(log a)
