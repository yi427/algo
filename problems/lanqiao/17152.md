# 17152. 最大区间

## [题目描述](https://www.lanqiao.cn/problems/17152/learning/)

给定一个长度为 n 的序列 A₁，求 L, R 使 (R - L + 1) · min(Aₗ, Aₗ₊₁, ..., Aᵣ) 尽可能大，其中 min 表示最小值。

你只需要输出最大的值即可，不需要输出具体的 L, R。

**输入格式：**
- 输入的第一行包含一个整数 n
- 第二行包含 n 个整数，分别表示 A₁, A₂, ..., Aₙ，相邻两个整数之间使用一个空格分隔

**输出格式：**
- 输出一行包含一个整数表示答案

**样例输入：**
```
4
1 3 3 1
```

**样例输出：**
```
6
```

**数据范围：**
- 对于 40% 的评测用例，1 ≤ n ≤ 5000，1 ≤ Aᵢ ≤ 5000
- 对于所有评测用例，1 ≤ n ≤ 3 × 10⁵，1 ≤ Aᵢ ≤ 10⁹

## 解题思路

这是一个经典的**单调栈**问题，目标是找到以每个元素为最小值的最大区间。

**核心思想：**
- 对于每个元素 Aᵢ，找到它作为最小值时能扩展的最大区间 [L, R]
- 该区间的贡献为 Aᵢ × (R - L + 1)
- 使用单调栈维护一个递增序列，当遇到更小的元素时，弹出栈顶并计算其贡献

**算法步骤：**
1. 在数组末尾添加 -1 作为哨兵，确保所有元素都能被处理
2. 维护一个单调递增栈，存储 (值, 索引) 对
3. 遍历数组：
   - 当前元素小于栈顶时，弹出栈顶元素
   - 弹出时计算：栈顶元素 × (当前索引 - 左边界 - 1)
   - 左边界是栈中倒数第二个元素的索引（如果不存在则为 -1）
4. 将当前元素入栈

**为什么这样做是正确的？**
- 单调栈保证了栈内元素递增
- 当遇到更小元素时，栈顶元素不能再向右扩展
- 栈中倒数第二个元素是左边第一个比它小的元素
- 因此区间范围是 (左边界, 当前索引)

## 代码实现

```python
def solve(nums: list[int]) -> int:
    nums += [-1]
    res = nums[0]
    s = []

    for i, num in enumerate(nums):
        while len(s) and s[-1][0] > num:
            prev = -1 if len(s) < 2 else s[-2][1]
            res = max(res, s[-1][0] * (i - prev - 1))
            s.pop()
        s.append((num, i))

    return res


if __name__ == "__main__":
    n = int(input())
    a = list(map(int, input().split()))
    print(solve(a))
```

## 复杂度分析

- **时间复杂度**：O(n)，每个元素最多入栈和出栈各一次
- **空间复杂度**：O(n)，栈的空间
