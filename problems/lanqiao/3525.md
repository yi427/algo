# 3525. 公因数匹配

## 题目链接

[蓝桥杯 3525](https://www.lanqiao.cn/problems/3525/learning/)

## 题目描述

给定 n 个正整数 Aᵢ，请找出两个数 i, j 使得 i < j 且 Aᵢ 和 Aⱼ 存在大于 1 的公因数。

如果存在多组 i, j，请输出 j 最小的那组。如果仍然存在多组 i, j，请输出 i 最小的所有方案中 j 最小的那组。

**输入格式：**

输入的第一行包含一个整数 n。

第二行包含 n 个整数分别表示 A₁, A₂, ..., Aₙ，相邻整数之间使用一个空格分隔。

**输出格式：**

输出一行包含两个整数分别表示题目要求的 i, j，用一个空格分隔。

**样例输入：**
```
5
3 2 6 9
```

**样例输出：**
```
2 4
```

**数据范围：**
- 对于 40% 的评测用例，n ≤ 5000
- 对于所有评测用例，1 ≤ n ≤ 10⁵，1 ≤ Aᵢ ≤ 10⁶

## 解题思路

本题要求找到两个有公因数的数，我们提供四种从简单到优化的解法。

### 思路演进

1. **暴力解法**：双重循环枚举所有数对，使用 gcd 判断是否有公因数
2. **质因数分解优化**：对每个数进行质因数分解，用哈希表记录每个质因子最后出现的位置
3. **埃氏筛优化**：预处理每个数的最小质因数(SPF)，加速质因数分解
4. **欧拉筛优化**：使用线性筛法预处理 SPF，将时间复杂度降到 O(n)

## 代码实现

### 方法一：暴力解法

双重循环枚举所有数对，使用 gcd 判断。

**时间复杂度**：O(n² log max(A))

```python
from math import gcd


def solve(nums):
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if gcd(nums[i], nums[j]) > 1:
                return i + 1, j + 1
    return -1, -1


if __name__ == '__main__':
    _ = int(input())
    a = list(map(int, input().split()))
    print(*solve(a))
```

### 方法二：质因数分解优化

对每个数进行质因数分解，用哈希表记录每个质因子最后出现的位置。

**时间复杂度**：O(n√max(A))

```python
from math import gcd

def get_prime_factors(n):
    factors = []
    d = 2
    temp = n
    while d * d <= temp:
        while temp % d == 0:
            factors.append(d)
            temp //= d
        d += 1
    if temp > 1:
        factors.append(temp)
    return factors


def solve2(nums):
    f = {}
    res = float('inf'), float('inf')
    for i in range(len(nums)):
        factors = set(get_prime_factors(nums[i]))
        for fac in sorted(factors, key=lambda x: f.get(x, 0)):
            if fac in f:
                res = min(res, (f[fac] + 1, i + 1))
            f[fac] = i
    return res


if __name__ == '__main__':
    _ = int(input())
    a = list(map(int, input().split()))
    print(*solve2(a))
```

### 方法三：埃氏筛优化

预处理每个数的最小质因数(SPF)，加速质因数分解过程。

**时间复杂度**：O(max(A) log log max(A) + n log max(A))

```python
def get_spf(max_val):
    """
    使用埃氏筛法预处理并返回 [0, max_val] 范围内每个数的最小质因数 (SPF)
    """
    spf = list(range(max_val + 1))
    for i in range(2, int(max_val**0.5) + 1):
        if spf[i] == i:  # 如果 i 是质数
            # 将 i 的所有倍数的 SPF 设为 i
            for j in range(i * i, max_val + 1, i):
                if spf[j] == j:
                    spf[j] = i
    return spf


def solve3(nums):
    f = {}
    res = float('inf'), float('inf')
    spf = get_spf(max(nums))
    for i, num in enumerate(nums):
        while num > 1:
            prime_factor = spf[num]
            if prime_factor in f:
                res = min(res, (f[prime_factor], i + 1))
            f[prime_factor] = i + 1
            while num % prime_factor == 0:
                num //= prime_factor
    return res


if __name__ == '__main__':
    _ = int(input())
    a = list(map(int, input().split()))
    print(*solve3(a))
```

### 方法四：欧拉筛优化

使用线性筛法（欧拉筛）预处理 SPF，时间复杂度最优。

**时间复杂度**：O(max(A) + n log max(A))

```python
def get_spf_euler(max_val):
    """
    使用欧拉筛（线性筛）预处理并返回 [0, max_val] 范围内每个数的最小质因数 (SPF)
    时间复杂度: O(N)
    """
    spf = list(range(max_val + 1))
    primes = []  # 用于按顺序收集所有找到的质数

    for i in range(2, max_val + 1):
        if spf[i] == i:
            # 如果 i 没被筛过，那它就是质数
            primes.append(i)

        # 遍历当前已经找到的质数
        for p in primes:
            if i * p > max_val:
                break

            # p 一定是 i * p 的最小质因数
            spf[i * p] = p

            # 【欧拉筛的核心灵魂】
            # 如果 i 已经是 p 的倍数了，说明 i 包含的最小质因数就是 p。
            # 那么对于更大的质数 p'，合数 i * p' 的最小质因数应该是 p 而不是 p'。
            # 为了保证每个合数只被其最小质因数筛一次，必须在这里 break。
            if i % p == 0:
                break

    return spf


def solve4(nums):
    f = {}
    res = float('inf'), float('inf')
    spf = get_spf_euler(max(nums))
    for i, num in enumerate(nums):
        while num > 1:
            prime_factor = spf[num]
            if prime_factor in f:
                res = min(res, (f[prime_factor], i + 1))
            f[prime_factor] = i + 1
            while num % prime_factor == 0:
                num //= prime_factor
    return res


if __name__ == '__main__':
    _ = int(input())
    a = list(map(int, input().split()))
    print(*solve4(a))
```

## 复杂度对比

| 方法 | 时间复杂度 | 空间复杂度 | 适用场景 |
|------|-----------|-----------|---------|
| 暴力解法 | O(n² log max(A)) | O(1) | n ≤ 5000 |
| 质因数分解 | O(n√max(A)) | O(n) | 通用 |
| 埃氏筛 | O(max(A) log log max(A) + n log max(A)) | O(max(A)) | max(A) 较大 |
| 欧拉筛 | O(max(A) + n log max(A)) | O(max(A)) | 最优解 |
