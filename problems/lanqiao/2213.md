# 2213. 打折

## [题目描述](https://www.lanqiao.cn/problems/2213/learning/)

小蓝打算采购 `n` 种物品，每种物品各需要买 `1` 个。

附近一共有 `m` 个店铺。第 `i` 家店铺会在时间区间 `[si, ti]` 内打折，折扣率为 `pi`。对原价为 `b` 的物品，折后价格为 `floor(b * pi / 100)`，其他时间按原价购买。

每家店铺会出售若干种物品。小蓝可以选择某一天去采购全部物品，问最少需要花多少钱。

题目保证小蓝一定能买到需要的所有物品。

**输入格式：**
- 第一行包含两个整数 `n, m`
- 接下来依次包含 `m` 个店铺的描述。每个店铺先输入一行 `si, ti, pi, ci`
- 然后接 `ci` 行，每行输入两个整数 `aj, bj`，表示该店铺出售的第 `j` 种商品类型和原价

**输出格式：**
- 输出一行一个整数，表示最少需要花费的钱数

**样例输入：**
```text
2 2
1 2 89 1
1 97
3 4 77 1
2 15
```

**样例输出：**
```text
101
```

## 解题思路

这道题使用扫描线 + 懒删除堆。

把每个店铺的折扣区间拆成两个事件：
- 在 `si` 时刻进入折扣状态
- 在 `ti + 1` 时刻退出折扣状态

对于每一种商品，维护当前所有可选价格中的最小值。由于折扣区间会动态开始和结束，可以为每种商品维护：

- 一个最小堆，保存当前可能成为最小值的价格
- 一个计数表，配合堆做懒删除

扫描所有事件时，只会影响某一种商品的价格集合，因此更新当前总花费时，只需减去该商品旧的最小值，再加上新的最小值。整个过程中维护答案最小值即可。

## 代码实现

```python
from collections import defaultdict
from functools import cmp_to_key
import heapq


def solve(tasks: list[tuple[int, int, int, int, int]], prices: list[list[int]]) -> int:
    # task: (time, type, price, fee, idx)
    n = len(prices)
    prices[0].append(0)

    tasks.sort(key=lambda x: (x[0], -x[1]))

    f = [defaultdict(int) for i in range(n)]
    lazy_heap = [[] for i in range(n)]

    for i in range(n):
        for p in prices[i]:
            f[i][p] += 1
            heapq.heappush(lazy_heap[i], p)

    total_fee = 0

    for i in range(n):
        total_fee += lazy_heap[i][0]

    res = total_fee
    for _, type, price, fee, idx in tasks:
        n_p = price * fee // 100
        top = lazy_heap[idx][0]
        if type == 0:
            f[idx][n_p] += 1
            f[idx][price] -= 1
            heapq.heappush(lazy_heap[idx], n_p)
        else:
            f[idx][n_p] -= 1
            f[idx][price] += 1
            heapq.heappush(lazy_heap[idx], price)

        while f[idx][lazy_heap[idx][0]] == 0:
            heapq.heappop(lazy_heap[idx])
        total_fee = total_fee - top + lazy_heap[idx][0]
        res = min(res, total_fee)

    return res


if __name__ == '__main__':
    n, m = list(map(int, input().split()))
    tasks = []
    prices = [[] for _ in range(n + 1)]

    for i in range(m):
        t1, t2, c, j = list(map(int, input().split()))
        for k in range(j):
            idx, p = list(map(int, input().split()))
            tasks.append((t1, 0, p, c, idx))
            tasks.append((t2 + 1, 1, p, c, idx))
            prices[idx].append(p)

    print(solve(tasks, prices))
```

## 复杂度分析

- 时间复杂度：`O(E log E + E log n)`，其中 `E` 是事件总数，排序事件需要 `O(E log E)`，每次更新堆的代价为 `O(log n)`
- 空间复杂度：`O(E)`，用于存储事件、堆和计数表
