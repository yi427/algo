# 19716. 商品库存管理

## 题目链接

[蓝桥杯 19716](https://www.lanqiao.cn/problems/19716/learning/)

## 题目描述

在库存管理系统中，跟踪和调节商品库存量是关键任务之一。小蓝管理的仓库中有多种商品，这些商品根据类别和规格被存放在不同编号的货架上，编号范围从 1 至 n。

为了高效地监控和调整库存量，小蓝的管理团队设计了 m 个操作，每个操作涉及到一个特定的商品区间 [L, R]。执行这些操作时，区间内每种商品的库存量都将增加 1。然而，在某些情况下，管理团队可能会决定不执行某些操作，使得这些操作涉及及的商品区间内的库存量不会发生改变。

现在，管理团队需要一个评估机制，来确定如果某个操作未被执行，那么最终会有多少种商品的库存量为 0。对此，请你为管理团队计算出，每个操作未执行时，库存量为 0 的商品种类数。

**输入格式：**

第一行包含两个整数 n 和 m，分别表示商品的种类数和操作的个数。

接下来的 m 行，每行包含两个整数 L 和 R，表示一个操作涉及的商品区间。

**输出格式：**

输出共 m 行，每行一个整数，第 i 行的整数表示如果不执行第 i 个操作，则最终库存量为 0 的商品种类数。

**样例输入：**
```
5 3
1 2
2 4
3 5
```

**样例输出：**
```
1
0
1
```

**数据范围：**
- 对于 20% 的评测用例，1 ≤ n, m ≤ 5 × 10³，1 ≤ L ≤ R ≤ n
- 对于所有评测用例，1 ≤ n, m ≤ 3 × 10⁵，1 ≤ L ≤ R ≤ n

## 解题思路

本题使用差分数组和前缀和来高效计算每个操作未执行时的影响。

核心思想：
1. 使用差分数组统计每个位置被多少个操作覆盖
2. 如果某个位置被覆盖次数为 1，去掉唯一覆盖它的操作后，该位置库存为 0
3. 如果某个位置被覆盖次数 ≥ 2，去掉任意一个操作后，该位置库存仍 > 0
4. 如果某个位置被覆盖次数为 0，无论去掉哪个操作，该位置库存都为 0

算法步骤：
1. 构建差分数组，统计每个位置的覆盖次数
2. 统计覆盖次数为 0 的位置数量（永远为 0 的位置）
3. 对于每个操作 [L, R]，计算该区间内有多少位置的覆盖次数恰好为 1

## 代码实现

### 方法一：直接统计

```python
from itertools import accumulate

if __name__ == '__main__':
    n, m = map(int, input().split())
    ranges = [list(map(int, input().split())) for _ in range(m)]

    diff = [0] * (n + 1)

    for (l, r) in ranges:
        diff[l-1] += 1
        diff[r] -= 1

    nums = list(accumulate(diff))
    zero = -1 + nums.count(0)
    nums = map(lambda x: 0 if x > 1 else x, nums)

    prefix = [0] + list(accumulate(nums))

    for (l, r) in ranges:
        print(prefix[r] - prefix[l-1] + zero)
```

### 方法二：区间计算

```python
from itertools import accumulate

if __name__ == '__main__':
    n, m = map(int, input().split())
    ranges = [list(map(int, input().split())) for _ in range(m)]

    diff = [0] * (n + 1)

    for (l, r) in ranges:
        diff[l-1] += 1
        diff[r] -= 1

    nums = list(accumulate(diff))

    zero = -1

    for i in range(len(nums)):
        if nums[i] > 2:
            nums[i] = 2
        elif nums[i] == 0:
            zero += 1

    prefix = [0] + list(accumulate(nums))

    for (l, r) in ranges:
        sub = prefix[r] - prefix[l - 1]
        r_len = r - l + 1
        print(r_len - (sub - r_len) + zero)
```

## 复杂度分析

- 时间复杂度：O(n + m)
- 空间复杂度：O(n)
