# 21211. 数字配对

## 题目链接

[蓝桥杯 21211](https://www.lanqiao.cn/problems/21211/learning/)

## 题目描述

小蓝有一个长度为 n 的正整数数组 A = [a₁, a₂, ⋯, aₙ]。每当他从中选出一对 (aᵢ, aⱼ)，其中 i < j，他会得到它们的差 b = aⱼ - aᵢ。然而，他特别喜欢数字 1，所以他想知道在数组 A 中最多能同时取出多少对正整数 (aᵢ, aⱼ)，每个数最多被取一次，使得它们的差为 1。

**输入格式：**

输入的第一行包含一个正整数 n。

第二行包含 n 个正整数 a₁, a₂, ⋯, aₙ，相邻整数之间使用一个空格分隔。

**输出格式：**

输出一行包含一个整数表示答案。

**样例输入：**
```
6
2 1 3 6 4 7
```

**样例输出：**
```
2
```

**样例说明：**

其中一种方案：(a₁, a₃), (a₄, a₆)

提示：(a₁, a₂) 的差为 a₂ - a₁ = -1，不是小蓝喜欢的。

**数据范围：**
- 对于 40% 的评测用例，1 ≤ n ≤ 10³，1 ≤ aᵢ ≤ 10³
- 对于所有评测用例，1 ≤ n ≤ 10⁶，1 ≤ aᵢ ≤ 10⁶

## 解题思路

本题使用贪心算法求解最大配对数。

核心思想：
1. 将相同数字的索引分组存储
2. 对数字从小到大排序
3. 对于每个数字 num，尝试与 num+1 配对
4. 贪心策略：从后往前取索引，确保满足 i < j 的条件

算法步骤：
1. 使用哈希表记录每个数字出现的所有索引位置
2. 按数字大小排序
3. 对于当前数字 num 和 num+1，从它们的索引列表末尾开始配对
4. 如果 num 的索引 < num+1 的索引，则可以配对，计数加1
5. 否则，丢弃 num 的当前索引，继续尝试

## 代码实现

```python
from collections import defaultdict


def solve(nums):
    res = 0
    f = defaultdict(list)
    for (i, num) in enumerate(nums):
        f[num].append(i)

    for num in sorted(f.keys()):
        v = f[num]
        n = f[num + 1]
        while v and n:
            if v[-1] < n[-1]:
                res += 1
                v.pop()
                n.pop()
            else:
                v.pop()
    return res


if __name__ == '__main__':
    _ = int(input())
    numbers = list(map(int, input().split()))
    print(solve(numbers))
```

## 复杂度分析

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
