# 19721. 最大异或结点

## [题目描述](https://www.lanqiao.cn/problems/19721/learning/)

小蓝有一棵树，树中包含 N 个结点，编号为 0, 1, 2, ⋯, N-1，其中每个结点上都有一个整数 Xi。他可以从树中任意选择两个不直接相连的结点 a, b 并获得分数 Xa ⊕ Xb，其中 ⊕ 表示按位异或操作。

问小蓝可以获得的最大分数是多少？

**输入格式：**
- 第一行包含一个整数 N，表示有 N 个结点
- 第二行包含 N 个整数 X₁, X₂, ⋯, Xₙ，相邻整数之间使用一个空格分隔
- 第三行包含 N 个整数 F₁, F₂, ⋯, Fₙ，相邻整数之间使用一个空格分隔，其中第 i 个整数表示第 i 个父结点编号，Fi = -1 表示结点 i 没有父结点

**输出格式：**
- 输出一行包含一个整数表示答案

**样例输入：**
```
5
1 0 5 3 4
-1 0 1 0 1
```

**样例输出：**
```
7
```

**样例说明：**
选择编号为 3 和 4 的结点，x₃ = 3, x₄ = 4，他们的值异或后的结果为 3 ⊕ 4 = 7

**数据范围：**
- 对于 50% 的评测用例，1 ≤ N ≤ 1000
- 对于所有评测用例，1 ≤ N ≤ 10⁵, 0 ≤ Xi ≤ 2³¹ - 1, -1 ≤ Fi ≤ N, Fi ≠ 0

## 解题思路

本题使用贪心算法求解最大异或值，结合树的结构约束（不能选择直接相连的结点）。

**核心思想：**
- 从最高位开始贪心地尝试构造最大的异或值
- 对于每一位，尝试将结果的该位设为 1
- 检查是否存在两个不相邻的结点，它们的值在当前位的异或结果为 1

**算法步骤：**
1. 构建树的邻接表，记录每个结点的相邻结点
2. 找到所有数字的最高位
3. 从最高位到最低位遍历：
   - 尝试将当前位设为 1
   - 使用哈希表按掩码分组结点
   - 检查是否存在两个不相邻的结点满足异或条件
4. 如果找到满足条件的结点对，更新结果

**关键优化：**
- 使用掩码逐位构造，避免枚举所有结点对
- 通过哈希表快速查找目标值
- 只检查不相邻的结点对

## 代码实现

```python
from collections import defaultdict


def solve(nums, fathers):
    if not nums or max(nums) == 0:
        return 0
    n = len(nums)

    adj = [set() for _ in range(n)]
    for i, fa in enumerate(fathers):
        if fa != -1:
            adj[i].add(fa)
            adj[fa].add(i)

    highest_bit = max(nums).bit_length() - 1
    res, mask = 0, 0

    for bit in range(highest_bit, -1, -1):
        mask |= 1 << bit
        flag = res | (1 << bit)

        seen = defaultdict(list)
        for idx, num in enumerate(nums):
            seen[num & mask].append(idx)

        for idx, num in enumerate(nums):
            target = (num & mask) ^ flag

            if target in seen and any(j not in adj[idx] for j in seen[target]):
                res = flag
                break

    return res


if __name__ == "__main__":
    _ = input()
    arr1 = list(map(int, input().split()))
    arr2 = list(map(int, input().split()))
    print(solve(arr1, arr2))
```

## 复杂度分析

- **时间复杂度**：O(n × log(max(nums)))，其中 n 是结点数量，需要遍历每一位
- **空间复杂度**：O(n)，存储邻接表和哈希表
