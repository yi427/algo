# 19722. 砍柴

## [题目描述](https://www.lanqiao.cn/problems/19722/learning/)

小蓝和小乔正在森林里砍柴。它们有 `T` 根长度分别为 `n1, n2, ..., nT` 的木头。对于每个初始长度为 `n` 的木头，小蓝和小乔准备进行交替砍柴，小蓝先出手。

每次砍柴时，当前木头长度为 `x`，需要砍下一截长度为 `p` 的木头，其中 `2 <= p <= x` 且 `p` 必须为质数。当轮到某一方时 `x = 1` 或 `x = 0`，它就无法继续砍柴，它就输了。

请对每根木头判断小蓝是否能赢。如果小蓝赢请输出 `1`，如果小乔赢请输出 `0`。

**输入格式：**
- 第一行包含一个正整数 `T`
- 接下来 `T` 行，每行包含一个正整数，其中第 `i` 个整数为 `ni`

**输出格式：**
- 输出 `T` 行，每行包含一个整数，依次表示对于每一根木头的答案

**样例输入：**
```text
3
1
2
6
```

**样例输出：**
```text
0
1
1
```

**评测规模与约定：**
- 对于 `20%` 的评测用例，`1 <= ni <= 10^3`
- 对于所有评测用例，`1 <= ni <= 10^5`，`1 <= T <= 10^4`

## 解题思路

这是一个典型的博弈 DP。

设当前木头长度为 `i`，如果存在某个质数 `p` 满足 `p <= i`，并且砍掉后能让对手处于必败态，那么当前状态就是必胜态；否则是必败态。

状态转移可以写成：

```text
win(i) = 存在质数 p <= i，使得 win(i - p) = 0
```

由于题目需要处理多组询问，可以先用欧拉筛预处理 `1` 到 `10^5` 内的所有质数，再做记忆化搜索、递推或状态压缩。

## 代码实现

### 解法一：记忆化搜索

```python
from functools import lru_cache


def euler_sieve(n: int) -> list[int]:
    spfs = [True] * (n + 1)
    primes = []

    for i in range(2, n + 1):
        if spfs[i]:
            primes.append(i)

        for p in primes:
            if i * p > n:
                break
            spfs[i * p] = False
            if i % p == 0:
                break
    return primes


def solve(nums: list[int], primes: list[int]):
    # f: True -> 小乔
    #    False -> 小蓝
    @lru_cache(maxsize=None)
    def dfs(i: int, f: bool) -> bool:
        if i == 1 or i == 0:
            return f

        for p in primes:
            if p > i:
                break
            if dfs(i - p, not f) != f:
                return not f
        return f

    for n in nums:
        print(int(dfs(n, False)))


if __name__ == "__main__":
    N = 1_000_00
    primes = euler_sieve(N)
    n = int(input())
    arr = [int(input()) for _ in range(n)]
    solve(arr, primes)
```

### 解法二：1:1 翻译成递推

```python
from functools import lru_cache


def euler_sieve(n: int) -> list[int]:
    spfs = [True] * (n + 1)
    primes = []

    for i in range(2, n + 1):
        if spfs[i]:
            primes.append(i)

        for p in primes:
            if i * p > n:
                break
            spfs[i * p] = False
            if i % p == 0:
                break
    return primes


def solve(nums: list[int], primes: list[int]):
    # f: True -> 小乔
    #    False -> 小蓝
    @lru_cache(maxsize=None)
    def dfs(i: int, f: bool) -> bool:
        if i == 1 or i == 0:
            return f

        for p in primes:
            if p > i:
                break
            if dfs(i - p, not f) != f:
                return not f
        return f

    n = max(nums)
    dp = [[-1] * (n + 1) for _ in range(2)]
    dp[0][0] = 0
    dp[0][1] = 0
    dp[1][0] = 1
    dp[1][1] = 1

    for i in range(2, n + 1):
        for f in range(2):
            dp[f][i] = f
            for p in primes:
                if p > i:
                    break
                if dp[not f][i - p] != f:
                    dp[f][i] = int(not f)
                    break

    for i in nums:
        print(dp[0][i])


if __name__ == "__main__":
    N = 1_000_00
    primes = euler_sieve(N)
    n = int(input())
    arr = [int(input()) for _ in range(n)]
    solve(arr, primes)
```

### 解法三：状态压缩

```python
def euler_sieve(n: int) -> list[int]:
    spfs = [True] * (n + 1)
    primes = []

    for i in range(2, n + 1):
        if spfs[i]:
            primes.append(i)

        for p in primes:
            if i * p > n:
                break
            spfs[i * p] = False
            if i % p == 0:
                break
    return primes


def solve(nums: list[int], primes: list[int]):
    n = max(nums)
    dp = [0] * (n + 1)

    for i in range(2, n + 1):
        for p in primes:
            if p > i:
                break
            if dp[i - p] == 0:
                dp[i] = 1
                break

    for i in nums:
        print(dp[i])


if __name__ == "__main__":
    N = 1_000_00
    primes = euler_sieve(N)
    n = int(input())
    arr = [int(input()) for _ in range(n)]
    solve(arr, primes)
```

## 复杂度分析

### 记忆化搜索

- 时间复杂度：`O(n * pi(n))`，其中 `pi(n)` 表示不超过 `n` 的质数个数
- 空间复杂度：`O(n)`，用于递归栈和缓存

### 1:1 翻译成递推

- 时间复杂度：`O(n * pi(n))`
- 空间复杂度：`O(n)`

### 状态压缩

- 时间复杂度：`O(n * pi(n))`
- 空间复杂度：`O(n)`
