# 17117. 火车运输

## [题目描述](https://www.lanqiao.cn/problems/17117/learning/)

钢厂有一辆用于运送废旧钢材的火车，它具有两节车厢，其中车厢 1 的最大载重量为 `A`，车厢 2 的最大载重量为 `B`。现在一共有 `N` 件废旧钢材需要被运输，其中第 `i` 件钢材的重量为 `wi`。

为了最大化运输效率，车长想要一次运输走重量尽可能多的钢材，请你帮助车长计算出一次运输最多可以带走多大总重量的钢材。

每件钢材都是独立的不可分割的，只能被放置在某一节车厢中。在装载钢材时只需要考虑重量条件即可。

**输入格式：**
- 第一行包含三个整数 `N, A, B`
- 第二行包含 `N` 个整数，分别表示 `w1, w2, ..., wn`

**输出格式：**
- 输出一个整数表示答案

**样例输入：**
```text
3 10 8
8 9 1
```

**样例输出：**
```text
18
```

**样例说明：**
- 车厢 1 装入第 2 件和第 3 件钢材，车厢 2 装入第 1 件钢材

**评测规模与约定：**
- 对于 `30%` 的评测用例，`1 <= N <= 10`
- 对于所有评测用例，`1 <= N <= 200`，`1 <= wi, A, B <= 1000`

## 解题思路

这是一个二维容量限制的 0/1 背包问题。

每件钢材有三种决策：
- 放入车厢 1
- 放入车厢 2
- 不选这件钢材

设状态表示当前考虑到第 `i` 件钢材时，在剩余容量或当前容量约束下所能得到的最大总重量。由于每件物品只能选一次，因此三种写法本质上都是 0/1 背包。

## 代码实现

### 解法一：记忆化搜索

```python
from functools import lru_cache
from math import inf


def solve(nums: list[int], A: int, B: int) -> int:
    n = len(nums)

    @lru_cache(maxsize=None)
    def dfs(i: int, a: int, b: int) -> float:
        if a < 0 or b < 0:
            return -inf
        if i < 0:
            return 0

        return max(
            dfs(i - 1, a - nums[i], b) + nums[i],
            dfs(i - 1, a, b - nums[i]) + nums[i],
            dfs(i - 1, a, b),
        )

    res = dfs(n - 1, A, B)

    return int(res) if res != -inf else 0


if __name__ == "__main__":
    N, A, B = list(map(int, input().split()))
    a = list(map(int, input().split()))
    print(solve(a, A, B))
```

### 解法二：递推

```python
from functools import lru_cache
from math import inf


def solve(nums: list[int], A: int, B: int) -> int:
    n = len(nums)

    dp = [[[0] * (B + 1) for _ in range(A + 1)] for _ in range(n + 1)]

    for i in range(1, n + 1):
        w = nums[i - 1]
        for a in range(A + 1):
            for b in range(B + 1):
                res = dp[i - 1][a][b]
                if a - w >= 0:
                    res = max(res, dp[i - 1][a - w][b] + w)
                if b - w >= 0:
                    res = max(res, dp[i - 1][a][b - w] + w)
                dp[i][a][b] = res
    return dp[n][A][B]


if __name__ == "__main__":
    N, A, B = list(map(int, input().split()))
    a = list(map(int, input().split()))
    print(solve(a, A, B))
```

### 解法三：状态压缩

```python
def solve(nums: list[int], A: int, B: int) -> int:
    dp = [[0] * (B + 1) for _ in range(A + 1)]

    for w in nums:
        for a in range(A, -1, -1):
            for b in range(B, -1, -1):
                res = dp[a][b]
                if a - w >= 0:
                    res = max(res, dp[a - w][b] + w)
                if b - w >= 0:
                    res = max(res, dp[a][b - w] + w)
                dp[a][b] = res
    return dp[A][B]


if __name__ == "__main__":
    N, A, B = list(map(int, input().split()))
    a = list(map(int, input().split()))
    print(solve(a, A, B))
```

## 复杂度分析

### 记忆化搜索

- 时间复杂度：`O(N * A * B)`
- 空间复杂度：`O(N * A * B)`，用于递归栈和缓存

### 递推

- 时间复杂度：`O(N * A * B)`
- 空间复杂度：`O(N * A * B)`

### 状态压缩

- 时间复杂度：`O(N * A * B)`
- 空间复杂度：`O(A * B)`
